# @wagent/dashboard

WAGENT Web Dashboard Server & Client — Halaman admin berbasis web (React + Express) untuk memonitor, mengonfigurasi, dan berinteraksi secara visual dengan AI Agent WAGENT.

## Panel Menu
- **Chat**: Chat panel real-time dengan status penyerahan obrolan (Human Takeover).
- **Nomor WA**: Hubungkan, putuskan, dan daftarkan beberapa nomor WhatsApp sekaligus secara live.
- **Persetujuan**: Antrean persetujuan manual (Approval Queue) untuk menyaring tindakan sensitif AI sebelum dikirim ke customer.
- **Knowledge Base**: Manajemen artikel FAQ bisnis dan unggah file dokumen untuk bekal RAG Agent.
- **Broadcast**: Buat jadwal dan kirim pesan massal otomatis.
- **Analytics**: CSAT tracker dan performa respons AI.

## Instalasi
```bash
npm install @wagent/dashboard
```
